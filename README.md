# CALM – Cinematic Advanced Lighting Matcher

プロフェッショナル向け照明ツール

## 概要

**CALM**は、Gaffer、DOP（撮影監督）、Lighting Director、Lighting Programmerのために開発されたプロフェッショナル向け照明ツールです。

映画、ドラマ、CM、MV制作において、実際のフォトメトリックデータを活用しながら最適な照明機材を選定できるよう設計されています。

## 主な機能

- 🔦 **プロフェッショナル向け照明データベース** - 実測フォトメトリックデータを収録
- 📊 **露出・光量計算** - 正確な計算に基づいた意思決定
- 💡 **複数灯の光量合成計算** - 複数の照明機材の組み合わせを即座に計算
- 🌈 **色温度ミックス計算** - LED・HMI・タングステンの色温度調整をサポート
- 🎯 **目標F値からの逆算機材検索** - 必要なF値から最適な機材を検索
- 🔄 **代替機材の提案** - 利用可能な機材での代替案を提示
- ⚖️ **LED・HMI・タングステン比較** - 異なる照明方式を相互比較
- 🎬 **プリプロダクション・現場支援** - 企画段階から撮影現場での意思決定をサポート

## CALMの理念

従来の照明計画では：

> 「このライトはどれくらい明るいのか？」

という発想でしたが、CALMは：

> 「この結果を得るためには、どのライトが必要か？」

を素早く導き出すことを目的としています。

これにより、限られた時間と予算の中で、より効率的で創造的な照明選定が可能になります。

## 対象ユーザー

- Gaffer（照明技師）
- DOP（撮影監督）
- Lighting Director（照明演出家）
- Lighting Programmer（照明プログラマー）
